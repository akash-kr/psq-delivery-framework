# Static Site Example

Use this adapter for simple HTML/CSS/JS sites or documentation sites where opening files or serving a static directory is enough.

Recommended harness commands:

```json
{
  "commands": {
    "install": "",
    "lint": "",
    "typecheck": "",
    "build": "",
    "test": "",
    "screenshot": ""
  }
}
```

Recommended proof:

- changed files
- browser screenshot if visual
- console error check if JavaScript exists
- manual notes for anything not automatable
